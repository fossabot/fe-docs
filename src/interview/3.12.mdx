7. vuex的mutation跟action，mapState
```
mutation和action的区别见310的第4问。
mapState: 当一个组件需要获取多个状态的时候，单独声明这些属性为计算属性会略显重复和冗余，可以使用mapState来辅助生成。
```
```javascript
<script>
  export default {
    computed: mapState([
      'count'
    ])

    computed: mapState({
      count: state => state.count
    })

    computed: {
      prevState() {},
      ...mapState({ ... })
    }
  }
</script>
```
